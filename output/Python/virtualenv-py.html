virtaulenv 使用

virtualenv可以将每一个开发项目的运行环境隔离开来，使他们各自使用的库互不干扰

sudo pip install virtualenv
virtualenv .env
source .env/bin/activate

可以通过文本安装库,避免一个个手动安装

pip install -r requirements.txt
